# 🧠 DSA Practice — Python

Repositório dedicado à prática de **Estruturas de Dados e Algoritmos (DSA)** com Python.

O objetivo não é escrever código rápido. O objetivo é **pensar corretamente antes de implementar** — escolher a estrutura certa, entender o custo da decisão e decompor problemas com clareza.

---

## O que você vai encontrar aqui

Cada pasta é um exercício independente, com:
- Implementação em Python
- Escolha justificada de estrutura de dados
- Análise de complexidade Big O
- Exemplos de entrada e saída

## Mentalidade

> Qual estrutura representa melhor esse domínio?  
> Qual operação precisa ser rápida?  
> Quanto custa essa decisão?

Esse é o início do pensamento de engenheiro — não apenas de programador.
